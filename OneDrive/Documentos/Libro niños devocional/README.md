# Devocionales - "Un día con Dios"

Proyecto: devocionales infantiles para edades 8-12.

- Página pública: GitHub Pages
- Autor / Organización: devocionalespequesypadres-collab

Contenido:
- HTML y CSS principales en `devocionalpage.html/` (ahora `index.html`)
- Plantilla dinámica en JavaScript para añadir más días

Cómo ejecutar localmente:
1. Abre el directorio del proyecto (ej. `cd "OneDrive/Documentos/Libro niños devocional/código/dia 1/devocionalpage.html"`)
2. Inicia un servidor local: `python -m http.server 8000`
3. Abre: `http://localhost:8000/index.html`
